# Quick Start Guide - Standard KSA Grading System

## For Teachers: How to Use

### Step 1: Access Grade Entry
1. Go to "My Classes"
2. Find your class
3. Click **"Midterm"** or **"Final"** button

### Step 2: Automatic Setup
- System automatically creates 22 standard components
- No setup required!
- Components appear immediately

### Step 3: Enter Grades
1. See all students listed
2. Enter scores in each column
3. Grades calculate automatically
4. Click **"Save All Grades"**

## Standard Components (Auto-Created)

### Knowledge (40% of grade)
- **Midterm Exam** (60%) - 100 points
- **Quiz 1** (13.33%) - 100 points
- **Quiz 2** (13.33%) - 100 points
- **Quiz 3** (13.34%) - 100 points

### Skills (50% of grade)
- **Output 1, 2, 3** (13.33%, 13.33%, 13.34%) - 100 points each
- **Class Participation 1, 2, 3** (10% each) - 100 points each
- **Activity 1, 2, 3** (5% each) - 100 points each
- **Assignment 1, 2, 3** (5% each) - 100 points each

### Attitude (10% of grade)
- **Behavior 1, 2, 3** (16.67%, 16.67%, 16.66%) - 100 points each
- **Awareness 1, 2, 3** (16.67%, 16.67%, 16.66%) - 100 points each

## Managing Components

### To Edit a Component:
1. Go to **"Settings & Components"** tab
2. Find the component
3. Click **"Edit"** button (pencil icon)
4. Modify name, weight, max score, or passing score
5. Click **"Update Component"**

### To Delete a Component:
1. Go to **"Settings & Components"** tab
2. Find the component
3. Click **"Delete"** button (trash icon)
4. Confirm deletion

### To Add a Component:
1. Go to **"Settings & Components"** tab
2. Click **"Add New Component"** button
3. Fill in the form:
   - Select KSA Category
   - Select Subcategory
   - Enter Name
   - Set Max Score (default: 100)
   - Set Weight (%)
   - Set Passing Score (default: 75)
4. Click **"Add Component"**

## Grade Calculation

### Formula:
```
Step 1: Calculate category averages
- Knowledge Average = Weighted average of all knowledge components
- Skills Average = Weighted average of all skills components
- Attitude Average = Weighted average of all attitude components

Step 2: Calculate raw grade
Raw Grade = (Knowledge × 0.40) + (Skills × 0.50) + (Attitude × 0.10)

Step 3: Apply transmutation (x50+50)
Final Grade = (Raw Grade × 0.50) + 50
```

### Example:
If a student has:
- Knowledge: 85%
- Skills: 90%
- Attitude: 95%

```
Raw Grade = (85 × 0.40) + (90 × 0.50) + (95 × 0.10)
Raw Grade = 34 + 45 + 9.5 = 88.5

Final Grade = (88.5 × 0.50) + 50
Final Grade = 44.25 + 50 = 94.25
```

## Features

### Real-Time Calculation
- Grades calculate as you type
- No need to click calculate button
- Instant feedback

### Color Indicators
- **Green border**: Score meets passing score
- **Red border**: Score below passing score
- **Red shake**: Invalid score (exceeds maximum)

### Total & Average Columns
- Each category shows Total and Average
- Helps track student performance
- Easy to spot issues

### Keyboard Navigation
- **Arrow Right/Left**: Move between inputs horizontally
- **Arrow Up/Down**: Move between students vertically
- **Enter**: Move to next row
- **Tab**: Standard tab navigation

### Validation
- Prevents scores exceeding maximum
- Highlights invalid entries
- Shows error messages
- Blocks saving invalid data

## Tips & Tricks

### Quick Data Entry:
1. Click first input
2. Enter score
3. Press **Arrow Right** to move to next component
4. Press **Arrow Down** to move to next student
5. Repeat

### Bulk Operations:
- Use **"Calculate All"** button to recalculate all grades
- Use **"Export Template"** to download Excel template
- Use **"Import Grades"** to upload scores from Excel

### Customization:
- Edit component names to match your syllabus
- Adjust weights to fit your grading policy
- Add extra components for special assessments
- Delete unused components to simplify table

## Grading Scale

- **90-100**: Excellent (Green badge)
- **75-89**: Passed (Blue badge)
- **60-74**: Conditional (Yellow badge)
- **Below 60**: Failed (Red badge)

## Common Questions

### Q: Do I need to set up components for each class?
**A:** No! Components are created automatically the first time you access grade entry.

### Q: Can I change the component names?
**A:** Yes! Use the Edit button in Settings & Components tab.

### Q: Can I add more components?
**A:** Yes! Click "Add New Component" in Settings & Components tab.

### Q: What if I don't use all components?
**A:** You can delete unused components or leave them empty.

### Q: How do I change the passing score?
**A:** Edit the component and change the "Passing Score" field.

### Q: Can I export grades to Excel?
**A:** Yes! Click "Export" button in the grade entry page.

### Q: What happens if I enter a score higher than the maximum?
**A:** The system will highlight it in red and prevent saving until corrected.
