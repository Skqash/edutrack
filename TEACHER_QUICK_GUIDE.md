## 📋 How to Configure Your Grading System

### Step 1: Go to Configuration

```
Teacher Menu → Classes → [Your Class] → Configure Assessment Ranges
```

### Step 2: Set Quiz Configuration

```
Total Quiz Items:        100  (or any number you prefer)
Number of Quizzes:       5    (Q1, Q2, Q3, Q4, Q5)
Equal Distribution:      ✓    (checked - recommended)
```

**The system calculates**: 100 ÷ 5 = **20 items per quiz**

### Step 3: Adjust Exam Scores (If Needed)

```
Prelim Exam:     60 items
Midterm Exam:    60 items
Final Exam:      60 items
```

### Step 4: Review the Preview

On the right side, you'll see:

- ✅ Pie chart showing distribution
- ✅ How each quiz contributes to your grade
- ✅ Test calculator to verify

### Step 5: Save

Click **Save Configuration** - Done! ✅

## 📝 How to Enter Grades

### Step 1: Go to Grade Entry

```
Teacher Menu → Classes → [Your Class] → Enter Grades
```

### Step 2: The New Smart Form

You'll see a table with:

- ✅ Student names on the left (stay visible when scrolling)
- ✅ Dynamic quiz columns: Q1, Q2, Q3, Q4, Q5
- ✅ Exam score
- ✅ Skills components
- ✅ Attitude components
- ✅ Real-time calculations

### Step 3: Quick Entry

**Click any cell → Type score → Tab to next → DONE!**

```
Example:
Q1: 18    (max 20)
Q2: 19    (max 20)
Q3: 17    (max 20)
Exam: 55  (max 60)
Output: 85 (max 100)
... etc
```

System automatically:

- Normalizes scores (converts to 0-100)
- Calculates component averages
- Calculates final grade

### Step 4: See Real-Time Stats

At the top:

- **Class Average**: Current avg of all grades
- **Highest**: Best student's grade
- **Lowest**: Struggling student
- **Passed**: Number of passing students
- **Failed**: Number of failing students

### Step 5: Search for a Student

Use the search box: Type name → Only matching students show

### Step 6: Undo a Mistake

Changed wrong cell?

```
Click: Undo → Last change reverted
```

### Step 7: Save When Done

Click **Save All Grades** - All grades saved to database ✅

## 🎓 Understanding Your Grading System

### Final Grade Calculation

```
Final Grade = Knowledge (40%) + Skills (50%) + Attitude (10%)
```

### Knowledge (40% of final grade)

```
Knowledge = Quiz Average (40%) + Exam Average (60%)
```

- **Quizzes**: All your quizzes averaged together
- **Exams**: Midterm exam average
- **Example**: Quiz avg=90%, Exam avg=85% → Knowledge = 87%

### Skills (50% of final grade)

```
Skills = Output (40%) + Class Participation (30%) + Activities (15%) + Assignments (15%)
```

### Attitude (10% of final grade)

```
Attitude = Behavior (50%) + Awareness (50%)
```

### Philippine Grading Scale

```
1.0  →  90-100%   [Excellent]
1.25 →  85-89%    [Very Good]
1.5  →  80-84%    [Good]
1.75 →  75-79%    [Satisfactory]
2.0  →  70-74%    [Acceptable]
2.25 →  65-69%    [Passing]
2.5  →  60-64%    [Passing]
2.75 →  55-59%    [Passing]
3.0  →  50-54%    [Passing]
4.0  →  Conditional
5.0  →  Failed
```

## ❓ FAQs

### Q: Can I have 10 quizzes instead of 5?

**A**: Yes! Go to Configuration → Change "Number of Quizzes" to 10

### Q: Will changing quiz items affect entered grades?

**A**: No! All calculations are based on the configured max. Old grades remain accurate.

### Q: What if I made a grading mistake?

**A**: Click **Undo** to revert. Or just click the cell and re-enter the correct score.

### Q: Can I see individual student progress?

**A**: Yes! Click on Analytics → Scroll to "Student Grade Breakdown" table

### Q: How do I print grades for records?

**A**: Go to Analytics → Click **Print** → Your browser's print dialog

### Q: Why is student's final grade different from my Excel calculation?

**A**: The system uses proper normalization. For example:

```
Old way: 18/20 + 19/20 + 55/60 = 92/100 ❌ (wrong - different scales!)
New way: (18/20)×100 + (19/20)×100 + (55/60)×100 = 99.17 ✅ (correct!)
```

### Q: Can students see their grades?

**A**: Student portal coming soon! They'll get notifications when grades are entered.

## 🔧 Troubleshooting

### Problem: Quiz columns showing Q1-Q10 but I configured 5

**Solution**: Refresh your browser (Ctrl+R or Cmd+R)

### Problem: Grade entry not saving

**Solution**:

1. Check internet connection
2. Make sure all required fields filled (no red borders)
3. Try clicking Save again
4. Clear browser cache if persistent

### Problem: Calculation looks wrong

**Solution**:

1. Go to Configuration
2. Use Test Grade field to verify calculation
3. If different, adjust max items
4. Contact support if still wrong
