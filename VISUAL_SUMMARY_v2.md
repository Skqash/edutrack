## 🔄 Grade Calculation Flow

```
┌─────────────┐
│ Teacher     │
│ Enters:     │
│ Q1: 18/20   │
│ Q2: 19/20   │
│ Exam: 55/60 │
└──────┬──────┘
       │
       ▼
┌─────────────────────────┐
│ SYSTEM NORMALIZES       │
│ Q1: (18/20)×100 = 90%   │
│ Q2: (19/20)×100 = 95%   │
│ Exam: (55/60)×100 = 92% │
└──────┬──────────────────┘
       │
       ▼
┌─────────────────────────┐
│ CALCULATES KNOWLEDGE    │
│ Quiz Avg = (90+95)÷2=93%│
│ Knowledge = (93×40%) +  │
│            (92×60%)     │
│          = 92.4%        │
└──────┬──────────────────┘
       │
       ▼
┌─────────────────────────┐
│ COMBINES WITH SKILLS &  │
│ ATTITUDE                │
│ Final = (92.4×0.4) +    │
│         (88×0.5) +      │
│         (90×0.1)        │
│       = 89.76%          │
└──────┬──────────────────┘
       │
       ▼
┌─────────────────────────┐
│ CONVERTS TO GRADE SCALE │
│ 89.76% → 1.0 (Excellent)│
│          Grade Stored   │
└─────────────────────────┘
```
