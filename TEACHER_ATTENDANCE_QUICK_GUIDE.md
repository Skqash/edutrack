# 📚 Teacher's Quick Reference - Attendance System

## 🚀 Quick Start (30 seconds)

### **Take Attendance**
1. Click: **Attendance** → Select your **Class**
2. Check that **Date & Term** are correct
3. For each student:
   - Click **✓** (Present), **✗** (Absent), or **🕐** (Late)
   - Click **🖊️** pen icon → **Draw signature** → **Save**
4. Click **Save** button (blue) at bottom

### **Print Attendance Sheet**
1. After saving, click **View Sheet** button
2. Click **🖨️ Print** button
3. Use browser print (Ctrl+P) → Save as PDF

## ⚡ Step-by-Step Guide

### **STEP 1: Navigate to Attendance**
```
Dashboard → Sidebar Menu → Attendance
```

### **STEP 2: Select Your Class**
```
Attendance Management
├─ Choose Class Dropdown → [BSIT 1-A Intro to Computing ▼]
├─ Term: [Midterm ▼] or [Final ▼]
└─ Date: [04/14/2026] ← Today's date (change if needed)
```

### **STEP 3: Mark Each Student**

**Option A: Mark Individually**
```
For each student row:
  Name: Fernandez, Rodrigo
  ┌─ Click: [✓] for Present
  ├─ Click: [✗] for Absent  
  ├─ Click: [🕐] for Late
  └─ Click: [🖊️] to Capture Signature
```

**Option B: Quick Mark All**
```
Click one of these buttons at the top:
  [All Present] - Mark everyone present instantly
  [All Absent]  - Mark everyone absent
  [Clear]       - Clear all selections
```

### **STEP 4: Capture Signature (if Present/Late)**
```
When you click the pen icon [🖊️]:

📱 E-Signature Modal Opens:
  ┌─────────────────────────────┐
  │ E-Signature for John Doe     │
  │                             │
  │ [Canvas for Drawing Here]   │ ← Sign here
  │                             │
  │ [Clear] [Undo] [Cancel]     │
  │         [Save Signature]    │
  └─────────────────────────────┘

Actions:
  - Draw signature with mouse/pen/finger
  - [Clear] = Erase and start over
  - [Undo] = Remove last stroke
  - [Save] = Save and close (button turns blue ✓)
```

### **STEP 5: Save Attendance**
```
After all students are marked:

At the bottom of the page:
  [Cancel] [View Sheet] [Save] ← Click here

Result:
  ✅ Attendance saved for 12 student(s) on April 14, 2026
```

### **STEP 6: View Attendance Sheet**
```
Option A: Immediately after saving
  - Click: [View Sheet] button → Opens in new tab

Option B: Anytime from Attendance page
  - Attendance → History → Click [Sheet] icon

Attendance Sheet shows:
  ┌─────────────────────────────────┐
  │ CPSU Class Attendance Sheet      │
  │                                 │
  │ Subject: Intro to Computing     │
  │ Date: April 14, 2026            │
  │ Term: Midterm                   │
  │                                 │
  │ No. │ Name │ E-Signature        │
  │ ────┼──────┼──────────────      │
  │ 1   │ John │ [signature image]  │
  │ 2   │ Jane │ [signature image]  │
  │ 3   │ Jack │ [empty if absent]  │
  │                                 │
  │ [⚙️ Select Date/Term] [🖨️ Print] │
  └─────────────────────────────────┘
```

### **STEP 7: Print or Save Sheet**
```
On the Attendance Sheet:

Option A: Print to PDF
  1. Click [🖨️ Print] button
  2. In print dialog: Select [Save as PDF]
  3. Choose location and filename
  4. Click [Save]

Option B: Print to Paper
  1. Connect printer
  2. Click [🖨️ Print] button
  3. In print dialog: Select printer
  4. Click [Print]

Option C: View Different Date/Term
  1. Click [⚙️ Select Date & Term]
  2. Change Date or Term dropdowns
  3. Click [Load Sheet]
  4. View updated sheet with signatures from that date
```

## 💡 Pro Tips

### **Time Saver**
- Use **[All Present]** button if everyone is present
- Takes 2 seconds instead of clicking each student

### **Signature Capture**
- **Don't worry about perfect signatures**
- Just needs to be a unique mark to identify student
- Can use mouse, trackpad, touchpad, or stylus

### **Batch Operations**
1. Click **[All Present]** to mark everyone
2. Then click **[✗]** only for absent students
3. Faster than clicking all 30+ students individually

### **Modify Attendance**
- Didn't mark correctly? Just change the status and save again
- The system updates the record (no duplicates)

### **View Previous Sheets**
1. Go to Attendance page
2. Click ⚙️ on the sheet
3. Change the date backward
4. Click [Load Sheet]
5. View and print past attendance records

## 🆘 Troubleshooting

### **"Signature won't draw"**
- Click [Clear] to reset
- Try clicking directly in the white area first
- Refresh browser and try again

### **"Can't see student names"**
- Make sure the correct **Course** is selected
- Check that **Class** is assigned students
- Contact Admin if no students appear

### **"Signature disappeared after Save"**
- It's saved! Check the [View Sheet] button
- The signature is stored in the database
- Print the sheet to see it clearly

### **"Date field is blank"**
- It defaults to today's date
- You can change it to previous days only
- Cannot record future attendance

### **"Save button doesn't work"**
- Make sure at least 1 student is marked
- Check that status buttons show selection (blue)
- Try refreshing and marking again
