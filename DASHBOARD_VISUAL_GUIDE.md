## 🔘 BUTTON ACTIONS EXPLAINED

### Configure Button (⚙️)
**What it does**: Opens the Advanced Configuration UI
```
Click ⚙️ → Advanced Configuration Page
   ↓
   Set Quiz Items (1-500)
   Set # of Quizzes (1-10)
   Set Exam Max Scores
   Configure Skills Components
   Configure Attitude Components
   ↓
   See Real-time Preview Chart
   See Per-Quiz Calculation
   ↓
   [Save Configuration]
```

### Entry Button (⌨️)
**What it does**: Opens the Quick Grade Entry form
```
Click ⌨️ → Quick Grade Entry Page
   ↓
   See Real-time Statistics Bar
   Search Students
   Click Cell to Edit
   Type Score & Tab
   See Auto-Calculation
   See Sticky Headers
   ↓
   [Undo] [History] [Save]
```

### Analytics Button (📊)
**What it does**: Opens the Analytics Dashboard
```
Click 📊 → Analytics Dashboard
   ↓
   See 4 Metric Cards
   • Class Average
   • Passed Count
   • Failed Count
   • Grade Range
   ↓
   View Charts
   • Grade Distribution
   • Pass vs Fail Ratio
   ↓
   View Performance
   • Knowledge %
   • Skills %
   • Attitude %
   ↓
   See Student Breakdown Table
```

## 🐛 TROUBLESHOOTING VISUAL ISSUES

### Issue: Buttons not showing
- **Solution**: Refresh page (F5)
- **Check**: Make sure you're logged in as teacher

### Issue: Cards not aligned
- **Solution**: Zoom reset (Ctrl+0)
- **Check**: Browser window not too narrow

### Issue: Colors look weird
- **Solution**: Clear cache (Ctrl+Shift+Del)
- **Check**: Using modern browser (Chrome, Firefox, Safari, Edge)

### Issue: Table not scrolling
- **Solution**: Try on wider screen
- **Check**: On mobile, use landscape mode
