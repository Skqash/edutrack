# Grade Summary - Class Filter Quick Guide

## 🎯 Quick Overview

The Comprehensive Grade Summary now has a **class filter** that lets you view specific classes or all classes at once.

---

## 📍 Where to Find It

```
Navigate to: Grades → Comprehensive Grade Summary
```

Look for the filter dropdown in the top right corner of the page.

## 💡 Tips & Tricks

### Tip 1: Quick Switching
You can quickly switch between classes without going back to the grades page.

### Tip 2: Print Individual Class
1. Filter to one class
2. Click "Print Summary"
3. Only that class will print

### Tip 3: Share Filtered View
The URL includes the filter, so you can:
- Bookmark specific class view
- Share link to specific class

Example:
```
/teacher/grades/summary-detailed?class_id=6
```

### Tip 4: Mobile Friendly
On mobile, the dropdown becomes full-width for easy tapping.

## 🔧 Troubleshooting

### Issue: Dropdown doesn't show my class
**Solution**: Make sure you've entered grades for that class first.

### Issue: Page shows "No Grade Data"
**Solution**: 
1. Check if you selected the right class
2. Click "View All Classes" to see all options
3. Go to Grade Entry to add grades

### Issue: Filter resets when I navigate away
**Solution**: This is normal. The filter is page-specific.
