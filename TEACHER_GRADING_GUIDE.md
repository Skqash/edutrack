# NEW GRADING SYSTEM - QUICK START GUIDE FOR TEACHERS

## 🎓 What's Different?

The new KSA Grading System uses:
- **Midterm Grade: 40%** (based on exams, quizzes, and assessment)
- **Final Grade: 60%** (based on final exam and final assessment)
- **Overall Grade** = Midterm (40%) + Final (60%)

## 📋 Component Overview

### Knowledge (40% Weight)
Your students' understanding and knowledge:
- **Exams**: Preliminary, Midterm, Final exams
- **Quizzes**: 5 quiz scores (Q1-Q5)
- **Formula**: (Quiz Average × 40%) + (Exam Average × 60%)

### Skills (50% Weight)
Your students' practical abilities and performance:
- **Output**: Portfolio/outputs (3 scores)
- **Class Participation**: Interaction in class (3 scores)
- **Activities**: Classroom activities (3 scores)
- **Assignments**: Homework assignments (3 scores)
- **Formula**: (Output × 40%) + (CP × 30%) + (Activities × 15%) + (Assignments × 15%)

### Attitude (10% Weight)
Your students' behavior and engagement:
- **Behavior**: Discipline and conduct (3 scores)
- **Awareness**: Class awareness/consciousness (3 scores)
- **Formula**: (Behavior × 50%) + (Awareness × 50%)

## 🚀 How to Enter Grades

### Step 1: Open Grade Entry Form
1. Go to your Teacher Dashboard
2. Select a class
3. Click "Enter Grades (New System)"
4. Or navigate to: `/teacher/grades/entry-new/{classId}`

### Step 2: Enter Student Scores
The form shows a comprehensive table with sections:

```
KNOWLEDGE (40%)  |  SKILLS (50%)  |  ATTITUDE (10%)  |  GRADES
Exams  Quizzes   |  Output CP Act Assign  |  Behavior Awareness  |  Midterm Final Overall
```

**Enter your scores:**
- Each score should be between 0-100
- Enter decimal scores (e.g., 85.5) if needed
- Leave blank if not applicable

### Step 3: Review Auto-Calculations
The system automatically calculates:
- ✓ Component averages (Knowledge, Skills, Attitude)
- ✓ Midterm Grade (based on current entries)
- ✓ Final Grade (based on final exam entries)
- ✓ Overall Grade (Midterm 40% + Final 60%)
- ✓ Letter Grade (A+, A, A-, B+, B, B-, C+, C, C-, D+, D, D-, F)
- ✓ Grade Point (4.0 scale)

### Step 4: Save Grades
1. Review all entries
2. Click "Save All Grades"
3. Confirm the save action
4. System will validate and save automatically

## 📊 Example Calculation

**Student: John Doe**

### Knowledge Component

### Skills Component

- Activities: 92, 95, 90 → Average = 92.33

- **Skills Score = (85 × 0.40) + (87.67 × 0.30) + (92.33 × 0.15) + (85 × 0.15) = 87.25**

### Attitude Component
- Behavior: 90, 88, 85 → Average = 87.67
- Awareness: 88, 85, 90 → Average = 87.67
- **Attitude Score = (87.67 × 0.50) + (87.67 × 0.50) = 87.67**

### Midterm Grade
**Midterm = (87.95 × 0.40) + (87.25 × 0.50) + (87.67 × 0.10) = 87.54**

### Overall Grade
**Overall = (87.54 × 0.40) + (Final × 0.60)**
*Depends on Final Grade once entered*

## ⚠️ Important Notes

### Multiple Entries Per Component
Each skill and attitude component has **3 entry fields**:
- Use these to record scores at different times
- The system averages them automatically
- This helps track student progress

### What Each Number Represents
- **1st Entry**: Early semester assessment
- **2nd Entry**: Mid-period assessment
- **3rd Entry**: Late semester assessment

### Midterm vs. Final
- **Midterm**: Entered based on midterm exams and early assessments
- **Final**: Updated after final exams and final assessments
- **Overall = Midterm (40%) + Final (60%)**

### Validation Rules
✓ All scores must be 0-100
✓ Decimal values allowed (e.g., 85.5)
✓ System prevents invalid entries
✓ All fields for a student can be saved at once

## 🎯 Best Practices

1. **Enter scores consistently**: Use same scale for all components
2. **Update regularly**: Don't wait until end of grading period
3. **Review before saving**: Check for obvious errors
4. **Track student progress**: Use multiple entries to show improvement
5. **Communicate expectations**: Make grading criteria clear to students

## ❓ Common Questions

### Q: What if a student doesn't have all 3 component entries?
**A**: The system averages only the entered values. You can have 1, 2, or 3 entries.

### Q: Can I edit grades after saving?
**A**: Yes! Just re-enter the form, update values, and save again.

### Q: What's the difference between Midterm and Final?
**A**: Midterm (40%) is for mid-semester, Final (60%) is weighted more for overall grade.

### Q: How is the letter grade determined?
**A**: Automatically based on the Overall Grade using the grading scale table.

### Q: Can I change the calculation formula?
**A**: The system uses the KSA formula (K:40%, S:50%, A:10%). Please contact administrators for changes.
