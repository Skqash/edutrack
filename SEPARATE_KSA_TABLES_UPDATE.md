1. **Midterm KSA Table** - Shows midterm grades with KSA breakdown
2. **Final KSA Table** - Shows final grades with KSA breakdown
3. **Overall Summary Table** - Shows the final grade calculation

## Calculation Display

### Midterm Grade Calculation
```
Midterm Grade = (Knowledge Г— 40%) + (Skills Г— 50%) + (Attitude Г— 10%)
```

**Example:**
```
Knowledge: 85.00
Skills: 88.00
Attitude: 92.00

Midterm = (85.00 Г— 0.40) + (88.00 Г— 0.50) + (92.00 Г— 0.10)
        = 34.00 + 44.00 + 9.20
        = 87.20
```

### Final Grade Calculation
```
Final Grade = (Knowledge Г— 40%) + (Skills Г— 50%) + (Attitude Г— 10%)
```

**Example:**
```
Knowledge: 90.00
Skills: 91.00
Attitude: 95.00

Final = (90.00 Г— 0.40) + (91.00 Г— 0.50) + (95.00 Г— 0.10)
      = 36.00 + 45.50 + 9.50
      = 91.00
```

### Overall Final Grade Calculation
```
Overall Grade = (Midterm Г— 40%) + (Final Г— 60%)
```

**Example:**
```
Midterm: 87.20
Final: 91.00

Overall = (87.20 Г— 0.40) + (91.00 Г— 0.60)
        = 34.88 + 54.60
        = 89.48
```

## Technical Details

### Table Structure

**Midterm Table:**
- 6 columns: Student Name, K, S, A, Midterm Grade, Decimal Grade
- Shows midterm-specific data only

**Final Table:**
- 6 columns: Student Name, K, S, A, Final Grade, Decimal Grade
- Shows final-specific data only

- 8 columns: Student Name, Midterm, Midterm Contrib, Final, Final Contrib, Overall, Decimal, Status
